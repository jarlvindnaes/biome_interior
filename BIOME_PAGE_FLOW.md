## 🎯 User Journeys

### Journey A: Direct to Form (Fast Converters)
```
Scan QR → Land on Hero → Click "Get Instant Access" 
→ Scroll to Form → Fill & Submit → Done ✅
Time: ~2 minutes
```

### Journey B: Library First (Explorers)
```
Scan QR → Land on Hero → Click "Explore the Library" 
→ Browse 6 categories → Click "Browse Full Library" 
→ View Stykka platform → Return → Scroll to Form → Submit ✅
Time: ~5-10 minutes
```

### Journey C: Full Read-Through (Researchers)
```
Scan QR → Read Hero → Read What is BIOME 
→ Browse Library categories → Check Who It's For 
→ Read FAQ → Scroll to Form → Submit ✅
Time: ~8-15 minutes
```

---

## 📱 Mobile Experience

All sections stack to single column on mobile:
- Hero text reduces from 7xl → 5xl
- 3-column grids become 1-column
- Form fields stack vertically
- CTAs remain full-width for easy tapping
- All interactive elements >44px tap target

---

## ⚡ Performance Optimizations

- **Lazy loading:** Framer Motion scroll animations only trigger when in viewport
- **Code splitting:** Each section is a separate component
- **No heavy images:** Uses CSS gradients and SVG patterns instead
- **Optimized fonts:** System fonts (no web font downloads)
- **Small bundle:** ~167 kB first load JS
